# Anchor

**An agent that actually remembers what you tell it.**

*"It remembers what you're avoiding."*

You text it in short bursts through the day — thoughts, worries, plans, complaints. Every message is logged and tagged. When you ask for a **reflection**, Anchor finds recurring patterns across your whole history — especially commitments you keep making and never following through on — and streams back a specific, evidence-based reflection.

Chatbots forget you; notes apps remember everything but notice nothing. Anchor pays attention.
